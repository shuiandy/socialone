# SocialOne

SocialOne is a SPA(Single Page Application) that allows user to view and manage their social accounts in on page.

## Dependencies

- Axios
- Cookies-next
- Moment
- Next
- NextUI
- Next-themes
- React
- React-facebook-login
- React-icons
- React-instagram-oauth
- React-player
- React-pro-sidebar
- Recoil
- Sass
- Twitter-api-v2

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```
